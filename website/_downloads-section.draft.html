Notas de instalación

Windows — SmartScreen

Windows puede mostrar una advertencia al abrir el instalador porque el binario aún no tiene firma digital. Haz clic en «Más información» → «Ejecutar de todas formas». El código fuente es abierto y auditable.

macOS — Gatekeeper

La app no está firmada con un certificado de Apple Developer, así que la primera vez macOS dirá que «no se puede verificar el desarrollador». Es normal. Para abrirla:

  1. Arrastra Vault Local a tu carpeta Aplicaciones.
  2. Haz clic derecho sobre el ícono → Abrir.
  3. En el aviso, pulsa Abrir de nuevo. Solo hace falta una vez.

Si macOS la bloquea del todo, ve a Ajustes del Sistema → Privacidad y seguridad y pulsa «Abrir de todas formas». El código es abierto y auditable.

Linux — AppImage

Dale permiso de ejecución y ábrela: chmod +x Vault*.AppImage && ./Vault*.AppImage. O instala el .deb con sudo dpkg -i Vault*.deb.
